GIT y GITHUB

GITHUB

Es una plataforma que nos brinda la posibilidad como estudiantes de crear y de poder publicar nustros proyectos de forma gratuita para ser presentados como nuestro portafolio de presentacion; de trabajos a las diferentes empresas que seran nuestras posibles obciones laborales.

GITHUB

En estaplataforma nos da el plus de desarrollar nuestros proyectos, e ir realizando versiones de mejoras del mismo.
    ejemplo :
  • GITHUB version 1.1.2
  • GITHUB version 2.2.1

GITHUB

es basicamente una red social para desarrolladores, es un sitio genial para mostra tus proyectos adicional que te brida la posibilidad de ser notado por reclutadores poteciales.

GIT
  • Tenemos a GIT la cual nos ayuda a simplificar el proceso de trabajo con otras personas, es decir nos brida la posibilidad de que otras personas puedan verificar los avances y puedan a su vez realizar mejoras al proyecto al mismo tiempo y sin dañar los archivos de los demas, permitiendonos creando versiones o archivos de proyecto
  • Este a su vez realiza un seguimiento automatico es decir deja un registro de la version anterior para asi poder volver a la anterior, sin importar cuantas veces sea modificado.
    • Diferencia entre GIT y GITHUB
    GIT
    • Es una herramienta de control de versiones distribuida que puede gestionar el historial de codigo fuente de un proyecto.
    • GIT es una herramienta de codigo abierto que los desarrolladores instalan localmente para gestionar el codigo fuente.
    GITHUB
    • Es una plataforma basada en la nube construida al rededor de la herramienta GIT.
    • GITHUB es un servicio en linea al que los desarrolladores que utilizan GIT pueden conectarse y cargar o descargar recursos
